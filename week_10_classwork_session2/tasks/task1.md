# Task 1

In a group of up to 4, discuss the data to be collected for one of the following systems:

- Patient information system
- Video sharing app
- Game platform
- Library management system

To figure out what data to collect, think about the followings:

- the purpose of the data (why collect it?)
- who generates it (patinets, users, gamers)?
- the sensitivity and ethical considerations (especially in healthcare)
- the impact on later stages (storage, analysis, privacy)

For example a fitness tracking app, data to be collected are:

- activity data (steps, distance, workout duration)
- heart and biometric data (heart rate, sleep patterns, stress levels)
- location data (GPS routes)
- user profile (age, sex, weight)

Data to be collected for your chose system are:

- Books: title, author, ISBN, category/genre, publication year, copy count, shelf/location.
- Members: name, contact details, membership ID, join date, status (active/suspended).
- Loans/returns: member ID, book ID, checkout date, due date, return date, fines.
- Staff/audit: staff ID, actions (issue/return), timestamps for accountability.
